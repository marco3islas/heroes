Sobre Comic's

Esta aplicacion de comics me sirvio como una forma de explorar y experiementar el trabajar con peticiones HTTP y Api Rest, esta dividia en dos secciones una de nombre Marvel y otra simplemente Heroes.

La primera esta conectada directamente a la base de datos de Marvel Comics y en el buscador que esta en esa seccion puede solicitarse informacion sobre los diferentes personajes de esa empresa.

La segunda es una base de datos local, mucho mas reducida y que tambien tiene un buscador pero solo cuenta con pocos personajes.
